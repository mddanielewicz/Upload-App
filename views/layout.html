About this application

This is a group effort project aimed at using the full M.E.A.N stack to upload and share short length videos

For this project we will be using Angular for the front end, Express to set up our server, Mongo DB with the fsGrid addon and NodeJS.

Individuals working on this project

This project was started by Ryan, Malachi, Matthew, and Michele to show case some of the things we have learned during Code Louisville

This is a portion of our code where I am testing out Bootstrap 4.0 for a basic layout design

By making this page we hope to be able to load and stream short video snippets allowing for upload capabilities.

What we will be using for this project(This may change as the project continues.

  • MongoDB
  • ExpressJS
  • AngularJS
  • NodeJS
  • GridFS
